# End of Day Journal Update

Update vault journals based on today's accomplishments.

## Workflow

### 1. Check Vault Access
Verify mount at `~/my-vault`.

### 2. Gather Context
- `~/my-vault/Active Todos.md`
- `~/my-vault/Daily/YYYY-MM-DD.md` (today)
- Current week's Checkpoint file

### 3. Collect Accomplishments
Ask what was accomplished. Can infer from checked items in Active Todos.

### 4. Update Files
- **Daily Note**: Update Done section, add session log
- **Active Todos**: Mark completed items `[x]`
- **Weekly Checkpoint**: Add accomplishments under today

### 5. Confirm Updates
Summarize what was updated and tomorrow's pending items.
