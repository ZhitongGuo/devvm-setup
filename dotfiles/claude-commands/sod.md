# Start of Day Setup

Set up for a productive day by reviewing context and creating today's daily note.

## Workflow

### 1. Check Vault Access
Verify the vault is accessible at `~/my-vault`. If not, check the symlink and Google Drive sync.

### 2. Gather Context
Read these files:
- `~/my-vault/Active Todos.md` - Current priorities
- `~/my-vault/Daily/YYYY-MM-DD.md` (yesterday) - Carryover items
- `~/my-vault/Current Week.md` - Current checkpoint week

### 3. Check Week Files
Verify `Current Week.md` points to the correct week. If the weekly checkpoint file doesn't exist, create it from template.

### 4. Create Today's Daily Note
Location: `~/my-vault/Daily/YYYY-MM-DD.md`
Populate Plan section from Active Todos + yesterday's carryover.

### 5. Present Daily Summary
Show all priority levels from Active Todos, active diffs, blockers, and carryover items.

### 6. Offer Next Steps
Ask which task to start with.
